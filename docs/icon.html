Icon

<rf-icon name="star"></rf-icon>

Styles

  • --icon-color: #161616 (for icon fonts)
  • --icon-cursor: default
  • --icon-fit: contain (for image icons)
  • --icon-font-size: 20px
  • --icon-line-height: 20px
  • --icon-size: 20px

Slots

None

Properties

  • data (object): Arbitrary data storage

Attributes

  • concealed (boolean): Do not show, but leave in DOM flow
  • hidden (boolean): Do not show in DOM
  • filled (boolean): Fill variation on icon font
  • name (string): Reference to icon in Google Fonts collection
  • src (string): Source for image icon
  • weight (integer, 100 - 700): Weight variation on icon font

Events

  • adc-load: Provided image source has loaded

Methods

None

Parts

  • image: Image element to use as alternative icon
  • font: Paragraph element for icon font from Google Fonts

Dependencies

None

# Stack

Shows a single child at a time.

## Importing

``` html
<script src="https://cdn.jsdelivr.net/npm/rainforest-web-components@latest/components/stack.js" type="module"></script>
```

## Examples

``` html
<rf-stack>
  <p>One</p>
  <p>Two</p>
  <p>Three</p>            
</rf-stack>
```

## Slots

| Name | Description |
| --- | --- |
| (`default`) | Children elements to be stacked. |
